World

The default build of Hashkat permits 3 languages, 4 ideologies, 3 regions, 2 preference classes and 200 agent_types, although the default build configures only 2 agent types. To change the maximum number of elements, go to build.

In Hashkat the output values may be reported by number not name.

Languages

At present the name and number of languages cannot modified in Hashkat, however the add_weight may be. That is, you may have 0% speakers of a language. However, agents must speak at least one language, so one language must have a non-zero add_weight.

Tweet Types

Tweet type affects the scope of the signal's re-transmission, and may be a factor in another agent's decision to follow or retweet. At present tweet types may not be modified, although an agent may transmit 0% of a certain tweet_type (ie 0% musical tweets). At least one tweet_type must have a non zero add_weight.

name scope of retransmission
Tweet Type 0 plain within language
Tweet Type 1 humorous within language
Tweet Type 2 ideological within language and region
Tweet Type 3 musical unlimited
Ideology

From one to four ideologies may be specified in the default build of Hashkat. Add_weight may be set to 0 for all but one ideology. Names of ideologies may be changed below.
