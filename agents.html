Agent Types

The default build of Hashkat permits up to 200 agent_types. Four default agent_types are listed below. The defaults may be modified by the user, and additional similar types may be added, or the user may define their own agent_types.

Add_weight may be set to 0 for all but one agent_type.

Agent Type 0 Standard View
Agent Type 1 Celebrity View
Agent Type 2 Organization View
Agent Type 3 Bot View
