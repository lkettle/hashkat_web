Run

What we are doing here:

  1. gather form data: config_static_h.php and INFILE_yaml.php
  2. save it to user prefered place with user given name
  3. in a separate div, ask user if wants to run a simulation
  4. give user a choice of saved configurations
  5. ask user where she wants to save output
  6. RUN, which means submit the 2 forms to the server
  7. hashkat runs simulation
  8. progress bar at user end
  9. some estimate of memory needed
  10. some estimate of time needed
  11. done message
  12. send user to output area
