Settings are automatically saved when you leave the page.

Default maximum numbers of each parameter are shown. Hashkat requires at least one Language, Region, Ideology, Agent Type and Preference Class have a non-zero add_weight. If you accept the default build, you may still assign an add_weight of zero to the parameter in your particular configuration. That is, you may have 0% speakers of a language.

*At present the number of languages and tweet types cannot modified in Hashkat.

Number of Languages* Add_weight may be set to 0 for all but one language.
Number of Tweet Types* Add_weight may be set to 0 for all but one tweet type.
Number of Ideologies Add_weight may be set to 0 for all but one ideology.
Number of Agent Types Add_weight may be set to 0 for all but one agent type.
Number of Preference Classes Add_weight may be set to 0 for all but one preference class.
Number of Regions Add_weight may be set to 0 for all but one region.
